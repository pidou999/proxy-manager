📖 使用指南
  1. 添加代理 — 支持 VLESS / VMess / Shadowsocks / Trojan / Hysteria2 / Tuic
  2. 导入/导出 — 批量导入链接（每行一个），导出全部链接
  3. 引擎控制 — 下载 sing-box → 启动 → 代理即生效
  4. 连接测试 — 点击 🧪 测试每个代理的连通性和延迟
  5. 每个分组可标记 ⭐ 默认出口，路由自动优先
  6. 🌍 全局代理 — 开启后写入系统环境变量，终端新会话自动走代理：
    • http_proxy=http://127.0.0.1:1081
    • https_proxy=http://127.0.0.1:1081
    • ALL_PROXY=socks5://127.0.0.1:1080
    • NO_PROXY=localhost,127.0.0.1,192.168.x.x

🔗 代理入口（重启后端口不变）

协议 地址 适用场景
SOCKS5 http://IP:1080 浏览器插件（SwitchyOmega）、终端
HTTP http://IP:1081 Docker 拉镜像、仅 HTTP 的应用
SOCKS5 本机 http://127.0.0.1:1080 宿主机本地应用、host 网络模式的容器
HTTP 本机 http://127.0.0.1:1081 宿主机本机 HTTP 请求

💡 所有端口监听在 0.0.0.0，局域网内任何设备把 IP 换成对应设备的地址即可使用。
⚠️ 无认证，请仅在可信内网使用。如要公网访问请配置防火墙或加密码。
